二. 关联github

1.注册一个githun帐号 并创建一个仓库

登陆过后点击new repository
仓库名为username/username.github.io 其中username为你github的帐号名称

2.添加SSH Key

执行以下命令获取密钥，密钥所在目录为(/c/Users/you/.ssh/id_rsa)：

ssh -keygen -t rsa -C “your_email@example.com” // 一直回车直至密钥生成成功
将公钥上传到github

如果同时使用github和gitlab 需要配置一下两个密匙
我就偷懒 贴一下教程链接
http://onthink.com/post/github-multi-account-configuration.html

访问username.github 是不是已经变成hexo博客了?
